GIT flow

GIT clone - для того, щоб "нахилити" репозиторій

GIT pull - для того, щоб отримати зміни з репозиторію

GIT checkout -b branchname - створити нову гілку та переключитися на неї

GIT add . - додати зміни, які зробили локально

GIT commit -m "commit text" - підписати їх

GIT push origin branchname - запушити на git сервер
